// Quick Install Guide
1. Upload all files to your webserver

2. Set folder rights:
	/media/files	chmod 777
	/media/images	chmod 777
	/media/txt		chmod 777
	/system			chmod 777

3. Open a browser and run your url to bloofoxCMS, e.g. www.your-domain-name.com/cms
	Install process will start automatically.
   
4. Delete folder /install from server

5. Delete folder /update from server

That's it! Read the documentation for more details...
